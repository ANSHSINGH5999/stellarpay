# StellarPay User Guide

## What StellarPay does

StellarPay is a Stellar testnet remittance demo for instant, transparent, low-fee transfers.

## How to start

1. Open the home page.
2. Click `Create New Wallet` to generate and fund a testnet account.
3. Wait for Friendbot funding to complete.
4. Open the dashboard to view balance and recent transactions.

## How to send money

1. Go to `/send`.
2. Enter an amount in INR.
3. Paste the recipient Stellar address.
4. Review the fee breakdown.
5. Confirm the transfer.
6. Open the resulting transaction in Stellar Expert.

## How to receive money

1. Go to `/receive`.
2. Copy your wallet address.
3. Share the address with the sender.
4. Refresh the dashboard after the payment arrives.

## How operators should use the ops center

1. Open `/ops`.
2. Review the verified user registry.
3. Check DAU, transaction totals, and retention.
4. Review monitoring and security sections before submitting.
